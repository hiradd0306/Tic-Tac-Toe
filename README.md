# Tic-Tac-Toe
This repository contains Python-based Tic-Tac-Toe game using Terminal.

# Running Tic-Tac-Toe
Open Terminal and use the commands below line by line:
```
git clone https://github.com/hiradd0306/Tic-Tac-Toe.git
cd Tic-Tac-Toe
python3 simulator.py
```
